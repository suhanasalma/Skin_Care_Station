Retinol

Retinol is a vitamin A . For a younger looking skin we need retinol. It works as a anti aging ingredients so we need to start this at the age of 25 and regularly we need to use sunscreen. we should not use it with vitamin c.

Niacinamide

Niacinamide is a vitamin b3. This is for all skin type, it balance skin oil production. Again we should not forget to use sunscrenn. This is best to use with Retinol.

Vitamin C

Vitam c is a super anti Oxident, best for spot free skin and brighter skin. It should be used between 10 to 20% but 15% is best to use for a greater result with less side effect. It's best practise to use it in the day time but don't forget to use sunblock. As it said that don't use it with retinol or with Niacinamide
